### Changed

- **Cleaner, safer encode flow (TigerCloud → TigerTag).** The encode dialog now has a centred title and a permanent instruction above the readers, and can be closed at any time (including mid-burn to abort). Each reader is drawn as a TigerTag reader plate with a corner status LED (red = no chip, green = chip detected). Presenting a chip while the dialog is open no longer pops a spool side-card over it.
- **Unified header status icons.** TigerScale, TD1S and the RFID readers now share larger, consistent icons. The two RFID reader badges are replaced by a single TigerPod icon (red = no reader, green = connected); hovering reveals each reader and the ID of any chip presented.
- Storage: "Clear all" now protects locked slots — a spool in a locked slot stays put when you clear a rack.

### Fixed

- Fixed an issue where Bambu Lab RTSP cameras (X1C / X1E / P2S / H2x) stayed black on Windows because no video component was available; the required component is now bundled on every platform, so the camera works out of the box.
- Fixed an issue where the update-status tooltip showed raw formatting tags instead of plain text.
