# Tiger Studio Manager v2.7.1

A quick follow-up to v2.7.0: your friends' shared lists now show up properly, and buy buttons read the shop name more cleanly.

### Changed

- Buy buttons now show the shop's main domain instead of a long address — "eu.store.bambulab.com" simply reads "bambulab.com".

### Fixed

- Fixed an issue where a friend's shared lists didn't appear when viewing their profile.
