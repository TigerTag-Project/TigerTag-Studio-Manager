## 🎉 Tiger Studio Manager turns 2.0

A big milestone release — the printer table gets a serious upgrade, updating a chip is now a proper guided flow, and a whole batch of bugs are gone.

### Added

- **Preview column** in the Printers table — see the model that's on the bed for whatever's currently printing, live from each printer.
- **"Ends at" column** — the clock time the current job finishes (e.g. `21:23`), so you know when to come back. Sortable by soonest finish.
- **Printer tags** — label your printers just like you tag spools, right in each printer's side card.
- **Search & filters for printers** — the search bar now works in the Printers view; the filters become Brand · State (online/offline) · Tags.
- **Multi-select delete** — tick several spools (or printers) and remove them together, with a hold-to-confirm. Always-on tick column in the table, shift-click for a range, Select all.

### Changed

- **Guided chip update** — "Please update RFID" now opens a clear panel showing your reader(s) waiting: the right chip lights up green, a wrong chip lights up red with a warning, and the write only runs when everything matches. Never writes the wrong chip.
- **Blank chip → straight to setup** — scan a fresh TigerTag chip and the "+ Material" panel opens right away.

### Fixed

- **Filament colour edits reach the chip for real** — editing a colour then hitting Update used to leave the chip on its old colour; it's now written and reads back correctly.
- **Elegoo print progress** no longer jumps between wild values mid-print.
- **"Storage location"** highlights the spool's slot again when you jump to Storage.
- **The "not stored" panel** refreshes a spool's picture the moment you recolour it.
- **Snapmaker firmware guide** is now translated across all 9 languages.
