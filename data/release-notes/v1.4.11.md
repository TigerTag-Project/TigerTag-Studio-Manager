### Added

- **FlashForge: live integration.** Connect to a FlashForge printer over your local network and monitor it in real time, with an edge-to-edge MJPEG camera feed that handles the printer's single-client limit gracefully (with a Retry button on error). The 5-slot material station shows each slot's state — filled, configured-but-empty, or unconfigured. The serial number is auto-prefixed if you leave off the "SN".

### Changed

- **Redesigned inventory toolbar.** The Grid / Table / Storage selector moved to its own row below the search bar, so it keeps its full width regardless of the filters above. The search input now has a clear (✕) button that appears as soon as you type.
