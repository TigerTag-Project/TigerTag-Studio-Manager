### Changed

- **More reliable TigerTag reference data.** The catalogue data (brands, materials, aspects, types, diameters, units, versions) now comes from a single source shared by the inventory and the live printer integrations. It tries the TigerTag service first and falls back to a recent mirror when unreachable, so offline users still get their last cached copy, and a truncated or malformed download is rejected rather than overwriting good data. A fresh install ships with the data already bundled, so it skips unnecessary downloads on first launch.
- **Redesigned Bambu Lab filament edit sheet** to match the Snapmaker / FlashForge / Elegoo design: two rows only (Filament and Color), and picking a colour closes the colour sub-sheet automatically. The sheet is now titled "Edit filament".

### Fixed

- Fixed an issue where the Bambu Lab printer state badge showed a raw label instead of "Idle".
