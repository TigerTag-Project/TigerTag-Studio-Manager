### Added

- **Contextual "+ Add Rack" inside the "Spools not stored" panel.** When you have more unstored spools than free slots — i.e. you actually need more rack capacity — a prompt appears right inside the panel. It's hidden when there's still room to drag spools into existing racks.

### Changed

- **"+ Add Rack" moved into the main header.** The header "Add" button now says "Add Rack" when you're in Storage, so there's one consistent place to add things whichever view you're in.

### Fixed

- Fixed an issue where the Storage stats counted ghost spools, letting the filled count exceed total capacity (e.g. "130/117 slots"). Stats now only count spools whose rack and slot actually exist, so the numbers match what you see.
- Fixed an issue where deleting a rack left its spools ghost-assigned to a rack that no longer existed, inflating the stats. Deleting a rack now fully frees its spools, older orphans are auto-cleaned the first time you open Storage, and they appear in "Spools not stored" so you can re-assign them.
- Fixed an issue where the Cam view's empty state showed raw placeholder text instead of a translated message after switching between Cam and the printer grid.
