### Fixed

- Fixed an issue where the Filaments Grid and Table views flashed on every search keystroke, rebuilding every card or row and re-decoding every image. Searching now filters the existing cards in place, so images stay put and the filter feels instant. The same applies to the Brand / Material / Version filters and the TigerTag / TigerTag+ / TigerCloud tiles.
- Fixed an issue where the printer grid refreshed constantly when printers were offline, flashing several times a minute on a large setup; it now updates only the online/offline badge inside each card.
