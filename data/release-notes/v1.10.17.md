### Fixed

- Fixed an issue where Creality fan controls did not match each printer's hardware. The live control card showed part-cooling, case and side fan sliders on every model, but only the enclosed K-series (K1 and K2 families) has all three. Open-frame printers — Hi, Ender-3 V4, SparkX i7 — and any unidentified model now show only the part-cooling fan.
