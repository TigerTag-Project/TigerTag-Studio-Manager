### Added

- **Anycubic control panel now works in cloud mode too.** Homing, jogging, disabling the motors, the light, nozzle/bed temperatures, the fan, the speed mode and pause/resume/stop now reach cloud-connected printers, not just local ones. Temperature, fan, speed and pause/stop apply to the printer's active job, so use them while a print is running.

### Fixed

- Fixed an issue where changing an Anycubic filament slot's material or colour flickered back to the old value before settling; it now switches once, cleanly.
