### Added

- **Shop.** Added a Shop button in the sidebar and at the bottom of the Friends panel that opens the TigerTag store in the browser (RFID makers, merch and more). It shows a one-time notification badge until first opened, like the GitHub / 3D Files / Discord buttons.
- **Reworked tag editor.** Adding tags to a spool now filters your existing tags as you type (multi-select with checkboxes) and lets you create a new tag inline; selected tags appear as chips below the field. A pencil button opens a full tag-management editor.
- **Picture view for racks.** Added a per-slot display switch in Storage: each rack slot can show either the colour fill with the remaining-weight bar (default) or a picture tile — a large square showing the material's illustration, or its colour.
- **Rack drag-and-drop sounds.** Dropping a spool into a rack plays a snap sound; moving one back to "not stored" plays a softer downward cue.
- **Picture-mode gallery for public inventories.** Opening a friend whose inventory is public now lands directly on a picture-mode gallery of their racks.

### Changed

- The Printers view switch is now hidden when viewing a friend's inventory, since printers are not shared.
- When dragging a spool onto an occupied rack slot, the swap (⇄) arrow now appears on both spools that will trade places, not only the one underneath.
- Printer notifications (e.g. the Snapmaker firmware alert) now show the brand logo without the surrounding white circle.

### Fixed

- Fixed an issue where bicolour, tricolour and rainbow filaments rendered a square edge outside the colour dot in the table, details, grid cards and group cards; the swatch is now a clean circle everywhere.
- Fixed an issue where spools of the same brand and material but different colours (e.g. white, red, rainbow) were incorrectly grouped together.
- Fixed an issue where moving spools, adding a tag, or any background change rebuilt the entire Storage view and scrolled it back to the top.
- Fixed an issue where a grouped set of spools showed as multiple rows in the table instead of a single line, unlike its single card in the grid.
- Fixed an issue where the notifications tab's close handle briefly overlapped a side panel opened immediately after.

### Removed

- Removed the orange dot next to the friend currently being viewed; the accent bar already indicates it, and the dot could remain on the wrong friend.
