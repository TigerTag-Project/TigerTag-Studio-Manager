### Added

- **Bambu Lab and Elegoo printers now show up in the network scan.** Add printer → Bambu Lab → Scan network discovers Bambu printers on the LAN with no setup, and Add printer → Elegoo → Scan network discovers Elegoo printers (Centauri Carbon 2 and later). Each scan offers one-click add with the serial number, model, IP and name already filled in, plus a manual "Enter IP address" path and an "Add by IP" shortcut. With Creality, Snapmaker and FlashForge already shipping discovery, every supported brand now has it.
- The Storage hover tooltip on a rack slot now shows the spool's material image alongside its brand, material, colour, weight bar, coordinate and lock indicator (for TigerTag+ spools with a product photo).

### Changed

- **Printers are reported online only once the connection is really established** — after the first real report or heartbeat arrives, not the instant the network socket opens. A printer that's reachable but not yet responding now correctly stays offline.
- Elegoo: the MQTT credential field is now labelled "Access code" and is required, matching the printer's own network settings screen (factory default is still `123456`).
- Cleaner Printer Settings form: the section headers and divider lines are gone, so the form reads as one continuous block of connection fields.
- Viewing a friend's inventory now hides the "+ Scan" and "Add" buttons, since they can't act on someone else's collection; they reappear when you return to your own view.
- The header backend-health indicator uses a new 3D cloud icon.

### Fixed

- Fixed an issue where re-scanning a spool wiped Firestore fields that weren't on the chip (container, note, capacity) and rolled the weight back to the chip's stale value. User-edited fields are now preserved, the weight is no longer rolled back, and when the database weight differs from the chip the app writes the new value onto the chip while it's on the reader.
- Fixed an issue where the password-eye and clear-input buttons jumped down when clicked, in Printer Settings, the login modal and the Add printer form.
- Fixed an issue where a Printer Settings input changed size when you clicked the eye toggle to show/hide a password.
