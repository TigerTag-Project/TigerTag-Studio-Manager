# Tiger Studio Manager v2.9.1

A quick fix so updates install properly on macOS — plus a second look at everything new in 2.9.0.

### Fixed

- Fixed an issue on macOS where clicking **Restart** after an update had downloaded only hid the window instead of installing the update, leaving the app running in the dock. Updates now install as expected.

> **On macOS, already on 2.9.0?** Quit the app once (⌘Q) and reopen it so 2.9.0's pending install completes — from there on, updates apply on their own.

### Also new since 2.9.0

If you missed it, 2.9.0 brought wishlist item quantities, a payment summary on every list, a "To order" cart grouped by store, fully offline QR codes (with colours and a download button), and a quick "I own a TigerPOD" switch. You'll see the full rundown inside the app.
