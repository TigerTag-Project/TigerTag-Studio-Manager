### Changed

- **Consistent view toggles.** The materials toggle is now Grid · Table · Storage, both toggle groups share the same Grid and Table icons and translations, and the printer "Cam" label is now localised.

### Fixed

- Fixed an issue where TigerCloud spools showed a wrong "Manufactured" date (around 2056). New spools are created with the correct date, and already-created spools are corrected on display and when later burned to a physical chip.
- Fixed an issue where a linked (twin) spool was counted twice in the "not stored" list, the not-stored count, the free-slot count and each rack's header, causing over-capacity totals. A twin pair now counts once, and auto-fill no longer scatters its two tags into separate slots.
