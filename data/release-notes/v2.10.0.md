# Tiger Studio Manager v2.10.0

A reorganised wishlist experience — drag-and-drop everywhere, clearer list types, and a few overdue fixes.

## New

- Wishlists can now be reordered by drag-and-drop, and dragged between the Private, Friends, and Public groups to change their type on the fly.
- Every product now has an info button that opens a quick side panel to set its buy link, price, and minimum stock, right next to the product itself.

## Improved

- Opening a list now shows a clean "Details" card up top — type, item count, event, and message — alongside a redesigned header.
- Private, Friends, and Public lists each get their own colour and badge, shown consistently across the sidebar, list details, and the create/edit screen.
- Your lists in the sidebar are now grouped by type, with "Create a list" pinned to the top.
- Creating or editing a list starts with a simple 3-choice type picker; new lists default to Private.
- In the "To order" view, clicking a material now opens its product card directly.
- Wishlist rows are more compact, and long shop names no longer spill out of their buttons.

## Fixed

- Fixed an issue where the Cam view showed the same Creality camera feed for every connected Creality printer instead of each one's own.
- Fixed an issue where a wishlist with many items couldn't be scrolled.
- Fixed an issue where two spools could end up assigned to the same storage slot; this is now detected and repaired automatically.
