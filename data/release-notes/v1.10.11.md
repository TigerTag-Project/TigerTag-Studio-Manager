### Added

- **Anycubic: file management.** The Anycubic printer panel now has a Files button that opens a browser for the printer's stored files. Browse on-printer storage and a USB stick (the USB tab appears once a stick is detected), and — for cloud-connected printers — a Cloud tab listing files you saved to Anycubic Cloud, with thumbnails. From any tab you can start a print or delete a file (both press-and-hold to confirm). Cloud files are shared across all your printers, so a sliced file is only printable on the model it was made for: incompatible files show which printer they belong to and their Print button is disabled. Works on both LAN- and cloud-connected Anycubic printers. Tested on a Kobra 3 V2 and a Kobra X.
- **Printer brand logos.** The printer list, grid and "add a printer" picker now show each manufacturer's logo (Bambu Lab, Anycubic, Creality, Elegoo, FlashForge, Snapmaker).

### Changed

- **Clearer online status.** A printer's online/offline state is now a small coloured dot next to its name — pulsing green when online, flat grey when offline — in both the grid and the side panel. The old "Online / Offline" badge has been removed as redundant.
- **"Last seen" instead of "Updated".** The printers list and grid now show when each printer was last seen online, remembered across restarts.
- **Tables remember your sort.** The filament and printer tables now keep your last sort column and direction across restarts. By default filaments sort by Brand and printers by status (online first).
- The printers table now has the same rounded, scrollable finish with a pinned header as the filament table.
- Connection labels now always note "(LAN)" for local connections (e.g. "WebSocket (LAN)", "MQTT (LAN)").
- Snapmaker setup: updated the Paxx U1 Extended firmware link and removed the now-unnecessary configuration step.

### Fixed

- Fixed an issue where a LAN-connected Anycubic printer's cooling fan read 0% and the nozzle/bed targets showed blank right after launch even when set; they now appear immediately on connect.
- Fixed an issue where the file browser's close button was rendered partly off the right edge of the window.
- Fixed an issue where the Bambu Lab filament/AMS card appeared empty when the printer was offline.
