## Added

- **A two-step welcome flow on first launch** — pick your language, then your display name and avatar (generate a colour or import a photo, with a crop step). Every account runs it once, including existing ones.
- **Custom links on a filament.** Attach a datasheet, a tutorial or a video to a product; the link sticks to every spool of that filament and shows in the spool details and the product card. Free accounts get one link, higher plans get more.
- **Racks can carry a subtitle** — a second line under the name, for what a single line never manages to say.
- **Filament colour can be edited again on TigerCloud spools.** A TigerCloud spool is fully digital, so its colour can be changed at any time; a written TigerTag chip stays read-only. No colour sensor required.
- **A "Product info" button on the spool details**, next to the list / cart / favourite actions, opening the product card beside it.
- **A clear button in the "Not stored" search field**, like the main search bar.

## Changed

- **The welcome screen you see before signing in has been rebuilt** — a new illustration of the whole system, a clearer line about what the app does for you, and three short points below the Sign-in button.
- **The sign-in window no longer opens on its own.** The app now lands on the welcome screen, and you open it when you want to.
- **"New rack" moved from a centred window to a side card**, like materials and printers, and is now laid out in three steps: name, layout, and a live preview of the rack. When you edit an existing rack, the preview shows what is actually in it.
- **Rack presets are gone** — set the number of levels and slots directly with − / + buttons. Faster than hunting for the preset that almost matches.
- **Racks now read from the top down**: the first slot, A1, is the top-left one, and new levels are added at the bottom.
- **The ✕ in a search field now replaces the magnifier** instead of appearing next to it, so nothing shifts around as you type.
- **Avatars: a new colour is generated for each new account**, and initials are built more sensibly from your display name — a single-word name now gives two letters instead of one, and accents are kept.
- **The avatar crop window handles like a desktop app**: zoom with the wheel or a trackpad pinch, free rotation, keyboard shortcuts, and double-click to reset.
- **In "To order", removing a line is now a single press-and-hold ✕** on the line. The text link only moves an item between the cart and the "saved for later" shelf, which now shows its own subtotal.
- **The "add to your to-order list" button only adds.** If the item is already there, it takes you to the list and tells you, instead of quietly removing it.
- **A new app icon**, and a new animated logo on the loading screen.

## Fixed

- Fixed an issue where a custom link added to a filament was not saved.
- Fixed an issue where the search in "Not stored" was cleared as soon as you moved a spool into a rack.
- A search that matches nothing now clears itself when you leave the field, instead of leaving you on an empty view with no explanation.
- Fixed an issue where the welcome flow showed your initials even though your account already had a photo.
- Fixed an issue where importing a photo and then generating a colour deleted the photo you had just uploaded.
- Fixed an issue where the image crop window opened behind the welcome window, which then looked frozen.
- Fixed an issue where the product card kept showing the previous product's information after you switched products.
- Fixed an issue where closing the "Add product" card left the colour picker floating on screen.
- Fixed an issue where a rack's subtitle did not appear until the view was reloaded.
- Fixed an issue where clicking a counter at the top of the window filtered the inventory and left the filter menu showing the wrong entry.
- Fixed an issue where "Add a price" did nothing in the Favorites grid.
- Fixed an issue where adding a Snapmaker U1 in LAN mode failed with a "save failed" message.
- Fixed an issue where the ✕ tab of a side card drifted away from the card while it slid open or closed.
- Fixed an issue where the product card's picture only updated after reopening it.
