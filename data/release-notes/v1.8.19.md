### Added

- **Printer connection tutorials.** Any brand with a tutorial (Bambu Lab, FlashForge, Elegoo) now shows a "Connection tutorial" button in the Add Printer panel. Pick your model from a visual grid and Studio walks you through each step with a screenshot and a one-sentence explanation, navigable with Prev/Next, the dots, or the arrow keys. Localised in all nine languages.

### Changed

- The printer table sorts by status by default, so connected printers come up first and offline ones at the bottom. Click any column header to sort differently.
- Cleaner printer cards: removed the grey rectangle behind the printer photo in the grid view, so the photo now sits directly on the card.
