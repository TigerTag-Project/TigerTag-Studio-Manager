### Added

- **Anycubic printers are now supported** — the sixth brand, alongside Bambu Lab, Creality, Elegoo, FlashForge and Snapmaker. Connect over your local network or through Anycubic's cloud (a mixed fleet works in one list), see the ACE multi-colour box and its slots, set a slot's filament (type and colour), and follow live job and temperature info, plus a camera feed on models that expose a local stream. Catalog: Kobra 3 / 3 Combo / 3 V2 / 3 Max / S1 / X. The Anycubic integration was contributed by [@ennisj](https://github.com/ennisj) (John Ennis).
- **Anycubic live control panel.** Drive a connected Anycubic printer straight from its side card, like Snapmaker and Elegoo: home the axes or disable the motors to move them by hand, jog X/Y/Z by 1 / 10 / 50 mm, set the nozzle and bed targets, toggle the light, control the part-cooling fan, and choose the print-speed mode (Silent / Standard / Sport). Each control shows an instant hover bubble mirroring Anycubic Slicer's own wording.
- **Anycubic filament management.** Each ACE slot reflects its real state: a present spool keeps its colour, while an empty slot shows a grey "?" with the colour kept as an outline so it stays recognisable. From a slot you can Load, Unload or Stop the filament feed, with each action enabled only when it applies. Editing a slot no longer makes the filament card flash; it updates only when the printer reports the change.
