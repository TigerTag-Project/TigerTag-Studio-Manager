### Added

- **Anycubic: printer error alerts.** When the printer refuses a command (for example "Home the axis before moving" if you jog before homing), an alert now shows the printer's message and error code, so you know why nothing happened.

### Fixed

- Fixed an issue where a cloud-connected Anycubic printer's fan, temperatures and speed mode only applied while a print was running; they now work at any time, using the same realtime channel as the official slicer.
- Fixed an issue where the Anycubic light toggled the camera light (which the printer rejects) instead of the chamber/part light.
- Fixed an issue where editing an Anycubic nozzle/bed temperature errored out; clicking away now applies the value, and Escape cancels.
