### Added

- **TigerScale: live panel.** Each TigerScale now has a live card with a Connect / Disconnect toggle and a large weight display with unit. It shows the current filament (colour, brand, material) as the scale reads it, a send-status badge reflecting the scale's state, and both reader slots side by side (a twin tag fills the second slot automatically). A one-second hold on TARE zeroes the scale, and a collapsible event log shows recent activity.

### Fixed

- Fixed an issue where Elegoo print-history thumbnails were silently dropped and never appeared.
