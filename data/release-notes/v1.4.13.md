### Added

- **Custom product image for DIY and Cloud spools.** A DIY or Cloud spool can now carry a product image from an external URL (TigerTag+ spools aren't editable). Set it from the colour square or the spool toolbox, and if the link ever breaks the app falls back to the colour placeholder and offers to edit it again. You can also set the image when adding a product.
- **Clear a spool's TD value.** A hold-to-confirm button on the "Scan TD" row lets you delete a spool's stored TD value.
- **TD1S sensor button when adding a product.** When a TD1S sensor is connected, scanning a filament auto-fills the colour and TD value fields.

### Changed

- New TigerCloud counter tile in the inventory header, and the DIY count now correctly excludes Cloud entries.
- **Refined window chrome.** A dark title bar on macOS and Windows, no OS drop shadow, and an update-status icon next to the health indicator (orange while downloading, green when ready — click it to install).

### Fixed

- Fixed an issue where the detail panel, filament edit sheet and printer side panel leaked a shadow outside the window while off-screen.
