## Removed
- Multi-select bar slimmed to **Delete + ×**: dropped the **"Clear"** button (redundant with ×, both called `_exitSelectMode`) and the **"Select all"** button (`bulkSelectAll` + its now-unused `_selectAllVisible` helper). Removed the button markup, handlers, and the `bulkClear` / `bulkSelectAll` i18n keys (9 locales); the remaining × icon button is now a perfect square (stretches to the Delete button's height + `aspect-ratio: 1`) — `renderer/inventory.html`, `renderer/inventory.js`, `renderer/locales/*.json`, `renderer/css/70-detail-misc.css`
- The detail panel's colour circle is no longer an edit trigger — clicking it used to open the Color & TD edit modal (with a hover "+" affordance). It's now a plain, non-interactive colour display; colour editing stays available via the other entry points (TD edit, TD1S measure, cloud encode). Dropped the `btnEditColor` click handler + `color-edit-trigger` / `color-edit-plus` markup, and their now-dead CSS — `renderer/inventory.js`, `renderer/IoT/td1s/td1s.css`
